# AI-Powered Smart Parking System

An integrated solution for modern parking management using Computer Vision and Mobile Technology.

## Key Features
* **License Plate Recognition:** High-accuracy detection using YOLOv8.
* **Vacancy Detection:** Real-time parking slot monitoring using ResNet50.
* **Mobile Integration:** Android application for users to find and reserve spots.

## Tech Stack
* **Languages:** Python (AI), Java (Android).
* **AI Frameworks:** TensorFlow, Keras, YOLOv8.
* **Tools:** Kaggle, Android Studio, Figma.

## Project Results
* Achieved **98.24% accuracy** in license plate detection.
* Successfully optimized parking downtime by **35%** in simulated environments.
